### 冒泡排序
- 比较次数为`O(n*n)`，交换次数为`O(n*n)`
- 代码
  ```
  for(int i = 0; i < arr.length - 1; i++) {
    for (int j = arr.length - 1; j > i; j--) {
      swap...
    }
  }
  ```

### 选择排序
- 比较次数为`O(n*n)`，交换次数为`O(n)`
- 代码
  ```
  for(int i = 0; i < length - 1; i++) {
    int min = i;
    for (int j = i + 1; j < length; j++ {
      swap...
    }
  }
  ```
