- 数组操作
- 两个数组可以按元素逐个执行二元操作符
```
a = array([1, 2, 3, 4])
b = array([2, 3, 4, 5])
a + b
a * b
a  ** b
```
- 单个数组
```
a + 1
a[0]
a[:2] + a[-2:]
```
- 形状
```
a.shape         -> (4L, )
a.shape = 2, 2  -> array([1, 2], [3, 4])
a * a           -> array([1, 4], [9, 16]) 多维数组乘法仍然是对应元素的乘积，并不是按照矩阵乘法来计算
```
- 生成等差数组： linspace(start, end, arr_size)
